#### npm
$ npm --verstion ( npm -v)
$ node --version ( node -v)
$ npm run test 파일명

#### git 명령어
$ git commit -m "메세지내용" 
$ git pull origin master 

##### branch
$ git branch -d 이름 ( 이름 branch 생성)
$ git branch 이름 ( 이름 branch로 들어가기 )
$ git -d 이름 ( 이름 branch 삭제, 강제는 -D)
$ git -m 이름 이름1 ( 이름을 이름1로 변경)

##### rollback 
$ git reset HEAD^ 이전 하나의 커밋 상태로 돌아가기
$ git reset HEAD^2 이전 2개의 커밋 상태로 돌아가기
