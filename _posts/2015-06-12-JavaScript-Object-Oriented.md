<!-- ## 理解对象

我们先创建一个 Object 实例，然后添加属性和方法，如下：

    person.sayName = function() {
        console.log(this.name);
    }

    var person = {

        job: "FE",
        sayName: function() {
            console.log(this.name);
        }
    };
